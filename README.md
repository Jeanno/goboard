# Front end go board library
Create a go board game for a user to place locally.
- Handles basic rules of go like capturing stones, ko
- Allow different board sizes
- Allow handicap stones
- Komi


# Design
- Should be mobile friendly.
- Looks good on both light and dark mode.


# Technology
- Typescript
- React
- Draw board using canvas
